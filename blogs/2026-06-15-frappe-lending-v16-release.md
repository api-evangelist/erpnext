Frappe Lending v16 ships with a redesigned versioning system aligned to ERPNext and Frappe Framework releases, and adds a new Loan Origination module alongside Loan Management so NBFCs can run complete lending workflows from lead capture through disbursement. Key additions include lead management, eKYC integration for Indian lenders, workflow automation with business rules, loan import tools, more flexible term loans, optional accounting features, and new loan outstanding, cashflow, and security ledger reports.
